About

This experiment attempts to find which artificial intelligence algorithm most acts like a human mind for solving a type of puzzle. The algorithms being tested for this experiment are a neural network algorithm, and a genetic algorithm. The humans will be the control group, and run the same puzzle as the algorithms will in a temperature regulated room.
